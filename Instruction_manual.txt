# 五教科学習・1対1学習クイズ　統合仕様書

## 1．概要

| 項目    | 内容                               |
| ----- | -------------------------------- |
| ジャンル  | 五教科の選択問題と、参加者同士の学習を組み合わせた学習クイズ |
| 参加形式  | 中学3年生同士による1対1の順番制の学習             |
| 基本形式  | 参加者は移動せず、交互に操作を出し合う             |
| 対象学年  | 中学3年生                            |
| 出題範囲  | 中学1年生から中学3年生の1学期まで               |
| 問題レベル | 中学1・2年生の内容も、中学3年生向けの難しさで出題する     |
| 教科    | 国語・数学・英語・理科・社会の五教科               |
| 実施の教科 | 実施の主催者が、実施開始前に一つの教科を選ぶ           |
| 完了条件  | 相手のポイントを0にした参加者の完了               |
| 初期ポイント  | 120                              |
| MP    | 使用しない                            |
| 選択肢   | すべて4択                            |
| 解答時間  | 操作1回につき20秒                        |
| 操作の数   | 4個                               |

---

## 2．学習の基本的な流れ

| 順番 | 内容                            |
| -: | ----------------------------- |
|  1 | 行動する参加者が、使用する操作を選ぶ           |
|  2 | 選んだ操作を確認し、決定する                 |
|  3 | 操作に対応する難易度の4択問題が始まる            |
|  4 | 20秒間で、できるだけ多くの問題に解答する         |
|  5 | 正解数と誤答数から有効得点を計算する            |
|  6 | 有効得点によって操作のポイント変動・加算量・保留効果が強化される |
|  7 | 操作の発動演出を行い、効果を反映する             |
|  8 | 相手参加者の順番に移る                |
|  9 | どちらかのポイントが0になるまで繰り返す            |

操作の選択時間は10秒とする。

---

## 3．得点の計算

### 有効得点

**有効得点 ＝ 正解数 − 誤答数 × 0.5**

有効得点が0未満になった場合は、0として扱う。

| 解答結果            | 得点への影響 |
| --------------- | -----: |
| 正解              |    ＋1点 |
| 誤答              |  −0.5点 |
| 無回答             |   変化なし |
| 時間切れ時に表示されていた問題 |    無回答 |
| 1問も解答しなかった場合    |  有効得点0 |

### 無回答の処理

* 無回答による追加ペナルティは発生しない。
* 1問も解答しなかった場合、操作の追加強化は発生しない。
* 操作の基本効果は、有効得点が0でも発動する。
* 1問以上解答したあと、次の問題を解いている途中で時間切れになった場合、その途中の問題は無回答として扱う。

---

## 4．使用できる操作

操作名は、特定の教科・属性・性別などに偏らず、効果が分かりやすい名前で統一する。

| 操作名   | 種類   | 効果                    | 効果の計算         | 待ち時間    |
| ---- | ---- | --------------------- | ------------- | ------- |
| 提出 | 通常提出 | 相手に安定したポイント変動を与える       | 8＋有効得点×4ポイント変動  | なし      |
| 確認 | 加算   | 自分のポイントを加算する            | 6＋有効得点×3加算    | 使用後1順番 |
| 保留  | 保留   | 次に受ける提出のポイント変動を減らす      | 15＋有効得点×7％軽減  | 使用後1順番 |
| 発展 | 発展操作  | 難しい問題に挑戦し、大きなポイント変動を与える | 16＋有効得点×8ポイント変動 | 使用後1順番 |

### 操作の追加ルール

* 保留の軽減率は最大60％とする。
* 保留の効果は、次に提出ポイント変動を受けるまで継続する。
* 確認を使用しても、ポイントは最大ポイントの120を超えない。
* 「使用後1順番」とは、使用した次の自分の順番では選べないことを意味する。
* 待ち時間が終了すると、再び使用できる。
* ポイント変動、加算量、軽減率に小数が出た場合は四捨五入する。

---

## 5．操作と問題難易度

| 操作    | 問題難易度 |
| ---- | ----- |
| 提出 | 通常    |
| 確認 | 通常    |
| 保留  | 通常    |
| 発展 | 難しい   |

発展は「発展操作」と「大操作」を統合した操作である。

難しい問題では、通常問題よりも次の要素を増やす。

* 複数の知識を組み合わせる
* 考える手順が多い
* 資料・表・グラフなどを読み取る
* よく似た選択肢から正確に判断する
* 基礎知識を応用して答える

出題範囲外の内容は使用しない。

---

## 6．五教科の問題方針

問題例や実際の問題文は仕様書には記載せず、どのような問題を出題するかだけを定める。

| 教科 | 出題する問題の種類                                  |
| -- | ------------------------------------------ |
| 国語 | 漢字の読み・意味、語句、文法、慣用句、ことわざ、短文読解、古文の基礎         |
| 数学 | 計算、方程式、比例・反比例、関数、図形、確率、資料の活用、式の展開・因数分解、平方根 |
| 英語 | 英単語、熟語、文法、空欄補充、語順、英文の意味、短い会話文、短文読解         |
| 理科 | 生物・化学・物理・地学の知識、実験、結果の予想、表・グラフ・図の読み取り、簡単な計算 |
| 社会 | 地理・歴史・公民、出来事の順番、地図、統計、年表、資料の読み取り           |

### 問題作成の共通ルール

| 項目         | 方針                           |
| ---------- | ---------------------------- |
| 問題形式       | 短時間で解答できる4択問題                |
| 正解数        | 選択肢のうち正解は一つ                  |
| 誤答選択肢      | 実際に間違えやすい考え方を反映する            |
| 問題文        | 必要以上に長くしない                   |
| 中学1・2年生の内容 | 単純な暗記だけでなく、中学3年生向けの判断・応用を含める |
| 通常問題       | 基礎知識と標準的な思考問題を中心にする          |
| 難しい問題      | 複数知識の組み合わせ、応用、資料読解を中心にする     |
| 公平性        | 両参加者で問題数や難易度に差が出ないようにする    |
| 解説         | 学習終了後に、正解と短い解説を確認できるようにする    |

---

# 7．ポイント変動演出

## 基本方針

実際に受けたポイント変動が大きいほど、次の演出を強くする。

* ポイント変動効果音
* キャラクターの赤色変化
* キャラクターの揺れ
* ポイント変動数字
* ポイントゲージの減少演出
* 特大ポイント変動時の一瞬停止

ポイント変動段階は、保留などによる軽減を計算したあとの最終ポイント変動で判定する。

---

## 8．ポイント変動の4段階

最大ポイント120を基準として、ポイント変動演出を4段階に分ける。

| 段階     | 最終ポイント変動 | 最大ポイントに対する割合 | 演出    |
| ------ | -----: | ---------: | ----- |
| 小ポイント変動  |   1～11 |      10％未満 | 控えめ   |
| 中ポイント変動  |  12～23 | 10％以上20％未満 | 標準    |
| 大ポイント変動  |  24～41 | 20％以上35％未満 | 強い    |
| 特大ポイント変動 |   42以上 |      35％以上 | とても強い |

同じ段階の中でも、ポイント変動が大きくなるほど音量・赤色・揺れ幅を少しずつ強くする。

---

## 9．ポイント変動演出の詳細

| 演出項目      | 仕様                        |
| --------- | ------------------------- |
| ポイント変動効果音   | ポイント変動が大きいほど音量を大きくする        |
| 大ポイント変動の音   | 大ポイント変動以上では、低い音や重い衝撃音を追加する  |
| 赤色変化      | ポイント変動を受けたキャラクターを0.25秒間赤くする |
| 赤色の濃さ     | ポイント変動が大きいほど赤色を濃くする         |
| キャラクターの揺れ | ポイント変動が大きいほど揺れ幅を大きくする       |
| 揺れ時間      | 最大0.4秒                    |
| 揺れる対象     | ポイント変動を受けたキャラクターだけ          |
| ポイント変動数字    | キャラクターの上に実際のポイント変動量を表示する    |
| 大ポイント変動の数字  | 数字を一度大きくしてから消す            |
| 特大ポイント変動    | 数字を大きく拡大し、一瞬停止と組み合わせる     |
| 一瞬停止      | 特大ポイント変動時のみ約0.1秒停止する        |
| ポイントゲージ     | 現在ポイントのゲージを先に減らす            |
| ポイントゲージの残像  | 減った部分を白色または黄色で一瞬残す        |
| 残像の動き     | 少し遅れて現在ポイントの位置まで縮小する        |
| 演出設定      | 開発時間を優先し、演出を弱める設定は実装しない   |

### 段階別の演出

| 演出   | 小      | 中      | 大      | 特大       |
| ---- | ------ | ------ | ------ | -------- |
| 効果音  | 小さな命中音 | 通常の衝撃音 | 大きく重い音 | 最も大きく重い音 |
| 赤色   | 薄い     | 標準     | 濃い     | 最も濃い     |
| 赤色時間 | 0.25秒  | 0.25秒  | 0.25秒  | 0.25秒    |
| 揺れ幅  | ごく小さい  | 小さい    | 大きい    | 最も大きい    |
| 揺れ時間 | 短い     | やや短い   | 最大0.4秒 | 最大0.4秒   |
| 数字   | 通常表示   | 通常表示   | 一度拡大   | 大きく拡大    |
| 一瞬停止 | なし     | なし     | なし     | 約0.1秒    |
| ポイント残像 | あり     | あり     | あり     | あり       |

### 演出上の注意

* 問題文や選択肢は揺らさない。
* 画面全体は揺らさない。
* 画面全体を赤く点滅させない。
* キャラクターが画面外にはみ出すほど揺らさない。
* 効果音が突然極端に大きくならないようにする。
* 特大ポイント変動の一瞬停止は、操作や得点計算には影響させない。

## 12．操作選択画面

| 表示内容   | 仕様                    |
| ------ | --------------------- |
| 操作名     | 大きく表示する               |
| 操作アイコン  | 操作の役割が分かるものを表示する       |
| 効果     | ポイント変動・加算・保留などを短く表示する   |
| 難易度    | 「通常」または「難しい」を表示する     |
| 待ち時間   | 使用できない操作に「あと1順番」と表示する |
| 使用不可の操作 | 暗く表示し、選択できないようにする     |
| 選択状態   | 選んだ操作の枠を明るくする          |
| 決定方法   | 操作を選んだあと、決定ボタンを押す      |
| 制限時間   | 10秒                   |

## 13．問題解答画面

| 表示内容  | 仕様                |
| ----- | ----------------- |
| 使用中の操作 | 画面上部に操作名とアイコンを表示する |
| 問題文   | 画面中央に大きく表示する      |
| 選択肢   | 押しやすい大きなボタンにする    |
| 残り時間  | 常に確認できる位置に表示する    |
| 正解数   | リアルタイム表示          |
| 誤答数   | リアルタイム表示          |
| 有効得点  | リアルタイム表示          |
| 操作ボタン  | 問題解答中は非表示にする      |
| 次の問題  | 解答後、自動的に切り替える     |

### 正誤表示

* 正解した場合、選んだ選択肢を一瞬明るくする。
* 誤答した場合、選んだ選択肢を一瞬暗くする。
* 誤答時は正しい選択肢も短く表示する。
* 表示後はすぐに次の問題へ移る。
* 正誤表示によって学習のテンポを大きく止めない。

---

## 14．操作発動画面

20秒の解答時間が終了したら、次の情報を短く表示する。

| 表示内容  | 例       |
| ----- | ------- |
| 正解数   | 正解：6問   |
| 誤答数   | 誤答：2問   |
| 有効得点  | 有効得点：5点 |
| 使用した操作 | 提出    |
| 計算    | 8＋5×4   |
| 最終効果  | 28ポイント変動  |

計算結果を表示したあと、キャラクターが操作を発動し、ポイント変動・加算・保留の演出へ移る。

---

## 15．加算と保留のUI表示

| 効果     | 表示方法                        |
| ------ | --------------------------- |
| 加算     | キャラクターの上に加算量を表示し、ポイントゲージを増やす  |
| 保留     | キャラクターの近くに保留状態を示すアイコンを表示する |
| 保留継続中 | 次に提出を受けるまでアイコンを残す           |
| 待ち時間中  | 操作ボタンに残り順番を表示する             |
| 待ち時間終了 | 操作ボタンを通常表示に戻す                |

# 17．全体の重要方針

* 学習問題の正解数が、操作の強さに直接つながる。
* 誤答には減点があるが、無回答には追加ペナルティを与えない。
* 問題を解くことと操作を選ぶことが複雑になりすぎないよう、MPは使用しない。
* 操作は4個に絞り、それぞれの役割を分かりやすくする。
* 発展操作の発展だけ、難しい問題を使用する。
* ポイント変動が大きいほど、音・赤色・揺れ・数字・ポイントゲージの演出を強くする。
* 問題文や選択肢は、戦闘演出の影響で読みにくくならないようにする。
* 色だけに頼らず、操作名・アイコン・数値でも情報を伝える。
* 学習終了後は正解と解説を確認でき、学習につなげられるようにする。
